XMPro Public Documentation

This repository contains the source files for the XMPro public documentation site, built using DocFX static site generator and deployed via Azure DevOps CI/CD pipeline.

🚀 Quick Start

For detailed setup and contribution instructions, see CONTRIBUTING.md.

# Install DocFX
dotnet tool install -g docfx

# Navigate to documentation folder
cd public-documentation

# Build and serve locally
docfx --serve

# View at http://localhost:8080

public-documentation/

🚢 Deployment Pipeline

How It Works

  1. Triggers on:

    • Push to main branch
    • Pull requests targeting main

  2. Build Process:

    • Builds DocFX documentation
    • Validates all internal links and images (external URLs are not checked)
    • Generates search index

  3. Deployment (main branch only):

    • Deploys to multiple version-specific GitHub repositories
    • Updates GitHub Pages sites automatically

Pipeline Configuration

  • azure-pipelines.yml - Main pipeline definition
  • templates/ - Reusable pipeline templates

Setup Requirements

  1. GitHub PAT Token:

    • Create a classic Personal Access Token
    • Grant repo or public_repo scope
    • Ensure write access to target repositories

  2. Azure DevOps Configuration:

    • Add PAT as DOCS_DEPLOY_PAT_CLASSIC variable
    • Mark as secret in pipeline settings

🔧 Configuration

Main Configuration

  • docfx.json - Production configuration
    • Includes all features
    • Used by CI/CD pipeline
    • Optimized build settings

Key Settings

{
  "build": {
    "content": ["**/*.{md,yml}"],
    "resource": ["images/**", "docs/**"],
    "output": "_site",
    "template": ["default", "modern", "xmpro-template"]
  }
}

🐛 Troubleshooting

Common Issues

  1. DocFX command not found:

    dotnet tool install -g docfx

  2. Port 8080 in use:

    docfx --serve -p 8081
