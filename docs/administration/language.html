Language

Overview

XMPro follows the common industry approach for how major no-code/low-code platforms handle translations for technical terms, i.e.:

  1. Keep in English:
  • SQL-related terms
  • Programming concepts
  • Industry-standard acronyms
  • Technical protocols
  1. Translate:
  • UI navigation elements
  • Error messages
  • Help documentation
  • Descriptive text

Certain things are in the primary company language: product roles & email templates.
