Key Conversion Patterns

Video Embeds

GitBook: {% embed url="https://www.youtube.com/watch?v=VIDEO_ID" %}
DocFX:   <iframe> HTML embed

Notes and Warnings

GitBook: {% hint style="info" %}
DocFX:   > [!NOTE]

GitBook: {% hint style="warning" %}
DocFX:   > [!WARNING]

Images

GitBook: <figure><img src="../../.gitbook/assets/Image Name.png"></figure>
DocFX:   ![Image Name](../../images/kebab-case-name.png)

Expected Issues and Solutions

Image Path Issues

  • Problem: Incorrect relative paths causing 404 errors
  • Solution: Use ../../images/ for files in /docs/blocks-toolbox/layout/

Image Naming Issues

  • Problem: Spaces in filenames cause rendering issues
  • Solution: Convert all to kebab-case (e.g., "Image Name.png" → "image-name.png")

GitBook Syntax Remnants

  • Problem: Hint blocks and embed tags remain
  • Solution: Convert to DocFX equivalents
